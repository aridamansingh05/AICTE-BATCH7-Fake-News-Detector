# Fake News Detection System

## Overview
This project detects fake news using Machine Learning (TF-IDF + Logistic Regression).
It also calculates confidence and credibility scores and supports optional live verification.

## Features
- Fake / Real classification
- Confidence score
- Credibility score
- Streamlit Web App
- NewsAPI verification (optional)

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run:
streamlit run app.py

## Dataset
Kaggle Fake and Real News Dataset
